Связи в базе данных

Category → Product | Одна категория содержит много товаров (FK: ЗАЩИТА)
User → Order | Один пользователь имеет много заказов (FK: ЗАЩИТА)
User → Transaction | Один пользователь имеет много операций (FK: ЗАЩИТА)
Order → OrderItem | Один заказ содержит много позиций (FK: ЗАЩИТА)
Product → OrderItem | Один товар в нескольких заказах (FK: ЗАЩИТА)
Order → Transaction | Один заказ имеет много операций (FK: ЗАЩИТА)
Order → OrderEvent | Один заказ имеет историю событий (FK: ЗАЩИТА, Только добавление)
